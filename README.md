# 🎮 Loja de Games
<br>
Este projeto consiste em uma API para gerenciamento de uma loja de games. A aplicação permite realizar operações completas de cadastro e controle de produtos e categorias, seguindo o padrão CRUD.

<br>

---

### 📌 Funcionalidades

A API oferece os seguintes recursos:

* **Cadastro:** Produtos e categorias.
* **Listagem:** Visualização completa de itens.
* **Busca Avançada:** Por ID, marca ou descrição.
* **Gestão:** Atualização e remoção de dados.

---

### 🛠️ Tecnologias Utilizadas

* **Linguagem:** TypeScript
* **Ambiente:** Node.js
* **Framework:** NestJS
* **Banco de Dados:** MySQL

---

### 🧱 Estrutura do Projeto

O projeto segue a arquitetura padrão do **NestJS**, organizada da seguinte forma:

* **Entities:** Representação das tabelas `Produto` e `Categoria` (Relacionamento 1:N).
* **Services:** Lógica de negócio e validações.
* **Controllers:** Gerenciamento das rotas e requisições HTTP.
* **Modules:** Organização modular do sistema.
